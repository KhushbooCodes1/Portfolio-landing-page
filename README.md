Responsive Portfolio Landing Page

This project is a personal portfolio landing page created using HTML, CSS and JavaScript. The purpose of this project is to showcase basic frontend development skills and create a responsive web interface.

## Features

- Responsive design for mobile and desktop
- Modern hero section with gradient background
- Skills section with colorful technology icons
- Projects section with hover animation
- Contact form with JavaScript validation
- Mobile responsive navigation menu

## Technologies Used

- HTML5
- CSS3
- JavaScript
